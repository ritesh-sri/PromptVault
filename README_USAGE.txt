IMPORTING PROMPTS
--------------------------------------------------------------------------------

BULK IMPORT FROM CSV:
   1. Go to File > Import > From CSV
   2. Select your CSV file
   3. Map columns (title, content, tags, platform)
   4. Click Import

CSV FORMAT EXAMPLE:
   title,content,tags,platform
   "Code Review","Review this code...","development,coding","ChatGPT"
   "Email Draft","Write an email...","writing,business","Claude"

IMPORT FROM TEXT FILE:
   1. Go to File > Import > From Text
   2. Each prompt separated by blank line
   3. First line = title, rest = content
   4. Select file and import
