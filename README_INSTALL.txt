INSTALLATION NOTES
--------------------------------------------------------------------------------

1. INSTALLATION DIRECTORY
   By default, PromptVault will be installed to:
   C:\Program Files\PromptVault

2. DATA STORAGE
   Your prompts database will be stored in:
   C:\Users\[YourName]\AppData\Roaming\PromptVault

   This ensures your data persists even if you upgrade or reinstall.

3. PERMISSIONS
   The installer requires administrator privileges to:
   - Install application files to Program Files
   - Create desktop and start menu shortcuts
   - Register keyboard hotkey (Ctrl+Shift+V)

4. FIRST RUN
   On first launch, PromptVault will:
   - Create a new SQLite database
   - Load starter prompts
   - Configure the global hotkey
   - Apply default settings
